# Dataset Directory

This folder contains dataset used in this project.

## Structure

* raw/        : Original LFW dataset
* processed/  : Preprocessed images (aligned, resized)
* embeddings/ : Face embeddings
* splits/     : Train/test split

## Note

Due to size limitations, dataset is NOT included in this repository.

You can download the dataset from:

* LFW dataset: https://www.kaggle.com/datasets/jessicali9530/lfw-dataset

After downloading, place the data into corresponding folders.
