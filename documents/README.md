## 🧠 Project Overview

This project focuses on building a **Face Recognition System** using:

* FaceNet (128-d and 512-d embeddings)
* Multilayer Perceptron (MLP) classifier
* Labeled Faces in the Wild (LFW) dataset

The system is implemented as:

* 🌐 Web Application (React + Flask)
* 💻 Desktop Application (Tkinter)
* 🎥 Real-time Webcam Recognition Module
