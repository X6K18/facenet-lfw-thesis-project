# 📊 Reports Directory

This folder contains all **reports, analysis results, and evaluation outputs** generated during the development of the Face Recognition System.

---

## 📚 Purpose

The `reports/` directory is used to:

* Store experimental results
* Present model evaluation metrics
* Provide analysis for FaceNet variants
* Support conclusions in the final thesis

---

## 📂 Structure

Typical contents include:

* `final_report.pdf`
  → Final thesis report

* `experiment_results.csv`
  → Raw results from model evaluations

* `metrics_summary.xlsx`
  → Summary of performance metrics

* `plots/`
  → Visualization charts (PNG, JPG)

* `comparisons/`
  → FaceNet128 vs FaceNet512 comparison results

---

## 🧪 Experiment Overview

This project evaluates:

* FaceNet128 (128-dimensional embeddings)
* FaceNet512 (512-dimensional embeddings)

The goal is to analyze:

* Accuracy differences
* Speed vs performance trade-offs
* Real-world applicability

---

## 📊 Evaluation Metrics

The reports include the following metrics:

* Accuracy
* Precision
* Recall
* FAR (False Acceptance Rate)
* FRR (False Rejection Rate)

---

## 📈 Visualizations

Generated charts may include:
